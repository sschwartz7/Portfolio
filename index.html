What I Use

REMEMBER TO CHANGE THIS I use C# on a daily basis to build full-stack webapplications. You can see my work implementing C# in my projects section!

I use proven design patterns and frameworks to build websites. ASP.NET Core Web and MVC fits the bill for most projects.

CHANGE THIS. I use HTML, CSS, and Bootstrap to build beautiful responsive websites. Bootstrap is my go to CSS framework. Many shy away from the front-end, but I embrace it.

CHANGE THIS. JavaScript is the language of the web. I use vanilla JS and I am confortable with many JS frameworks. Check out my challenges section for some JS projects

CHANGE THIS. SQL is the language of the database. My projects all utilize a database. I have used Postgres on all of them. I can write SQL on any DB platform
