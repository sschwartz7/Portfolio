What I Use

I use C# to build all of my full-stack webapplications. Check them out in my projects section.

I use proven design patterns and frameworks to build websites. ASP.NET Core Web and MVC fits the bill for most projects.

I have used a combination of HTML, CSS, and Bootstrap to build all of my websites. I enjoy using Bootstrap to create responsive well designed websites.

I have extensive knowledge of JavaScript, which I have used to build many projects. Check out my small project section for some JS projects

All of my large project utilize a database. While I can write SQL on any DB platform, Postgres is my database of choice.

All of my projects are stored and updated on GitHub using the git language. I use CI/CD to keep things up to date.
